Leveling my bed and managing with the screen

For today I am going to level my bed, there are three springs under my bed which holds it. I put my nozzle above each one of the springs and leveled it by using an A4 paper, it will be easier to see how high your nozzle is. Before doing this, I tried to print a voron cube but failed as the nozzle hit the bed for some parts and being too high for others. I bought the official screen for Voron0 and wanted to use it. I tried to find different configuration manuals but they were not the same as mine, I bought mine from BigTreeTech, when I went onto their official Github page, I couldn't find any reference files for it as they haven't put any yet. Because I couldn't find any files and I am not really into using it very often, I decided not to use it for this printer, maybe there can be other uses for it.
